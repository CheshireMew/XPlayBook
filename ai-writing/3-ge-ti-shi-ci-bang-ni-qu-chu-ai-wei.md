# 3 个提示词帮你去除 AI 味

1、提高原创性

{% code overflow="wrap" %}
```
请在保留原文主要思想的前提下，重新组织句子和段落，让内容逻辑更加顺畅自然。可以适当调整表达顺序或用词方式，使文章在结构和语言上都有新的呈现。这样既能提高原创度，又能让读者更容易理解你的思路。
```
{% endcode %}

2、 防止虚构与抄袭

{% code overflow="wrap" %}
```
请务必确保所有内容都基于事实，不允许编造信息。如果遇到不清楚的地方，就直接写“我不知道”，不要凭空想象。请用自己的语言表达，避免照搬别人的文字（引用内容需按照 MLA 格式标注）。
写作风格可以兼顾正式与自然，不要太直接也不要太随意。每个句子都要有主语，尽量使用短句，让表达更清晰。把生僻或书面化的词语换成日常常用词，用基本、自然的连接词保持句子之间的连贯。整个内容要逻辑清晰，语言简洁自然。
```
{% endcode %}

3、增加具体细节

{% code overflow="wrap" %}
```
在需要的地方扩展内容，加入具体的细节、案例或数据，让表达更有说服力。避免只写笼统的句子，可以举实际的例子或增加一些场景描述，使内容更加生动、形象、有画面感。这样能让文章读起来更吸引人，也能更好地展示你的理解深度。
```
{% endcode %}
