# 算法喜欢什么样的推文？

推文的评分主要基于以下几个特征：**文本长度、可读性、喊叫程度（Shoutness）、文本熵**、以及**链接**的使用。&#x20;

#### 以下是一些通俗易懂的相关建议：&#x20;

### 1.不要太长也不要太短&#x20;

推文就像写小作文一样，太长了会让人不想读，太短了又讲不清楚。写得刚刚好，就会得高分。这一点要向另维小姐姐[@GalaBNB](https://x.com/GalaBNB)学习。&#x20;

### 2.让你的句子有可读性&#x20;

可读性主要由词汇的平均长度和句子的复杂性决定。推文就像说话一样，别人能轻松理解你写的内容很重要。用简单的词，写清楚的句子，这样大家都能明白。&#x20;

关于如何做到这一点，请看这篇小故事《你成为更好作家的那一天》： [https://x.com/0xCheshire/status/1829298990413967423](https://x.com/0xCheshire/status/1829298990413967423)

### 3.少用全大写字母&#x20;

这一点是针对英语用户的，不清楚有没有特地对中文做什么措施。大写字母就像是在喊话，如果你写了一大段全是大写字母，读起来就像在大吼大叫，会让推文评分下降。&#x20;

### 4.多用不同的词&#x20;

使用丰富的词汇和句式可以提高熵值，从而提高推文的评分。不要老是用同样的词，多试试新的词。比如说第二次说“苹果”你可以说“林檎”（不是）&#x20;

### 5.适当地使用链接（最好是站内链接）&#x20;

推文中加上链接就像给作文附上参考资料。链接可以增加推文的价值，但过多的链接可能会被认为是垃圾信息，因此要控制链接的数量，确保它们是相关且有价值的。&#x20;

关于发外链的警醒：[https://x.com/0xCheshire/status/1829333453344854062](https://x.com/0xCheshire/status/1829333453344854062)

### 6.别说脏话或不礼貌的话&#x20;

如果你在推文里用脏话或不礼貌的词汇，评分会大大下降，就像老师会在你的作文上扣分一样。如果推文包含冒犯性内容或者用户名具有冒犯性，这些都会显著降低推文的评分。
